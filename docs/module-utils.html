(static) validateKey(key, keysopt) → {module:utils~KeyValidationResult}

Checks if the key is valid.

This function checks if the provided key is valid. For a key to be valid, it has to pass the following tests :

  • Must be a string.
  • Must not contain a separator, as defined by path.sep.
